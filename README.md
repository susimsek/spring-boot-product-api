# Technologies Used
- Java 8
- Spring Boot
- Spring MVC
- Spring Data JPA
- Apache Commons Logging
- Mysql 5.7
- Docker 18.09.2
- Docker Compose 1.23.2
- Spring Dev Tools

# Requirement for Development and Build
- JDK 1.8
- Maven 3.x
- Docker 18.x
- Docker Compose 1.2x

# Command for Run the Program
- docker-compose up

# Command for Stop the Program
- docker-compose down
